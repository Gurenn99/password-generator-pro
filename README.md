# Password Generator Pro

**A secure, customizable, professional-grade password generator built in Python.**

---

## Features

- Generate ***1-20** strong passwords
- Custom length: **8-50** characters
- Choose characters types:
  - Uppercase
  - Lowercase
  - Digits
  - Symbols
- **Excludes ambiguous characters** ('l', '1', 'O', '0', 'I')
- **Save passwords to 'passwrods.txt'**
- **Copy any password to clipboard** with one click
- Clean, user-friendly interface
